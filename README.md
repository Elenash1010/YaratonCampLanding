# YaratonCampLanding

Статический лендинг для детского лагеря и семейного курорта "Яратон".

## Структура

- `index.html` - единственная входная страница.
- `assets/images` - изображения, разложенные по тематикам.
- `assets/logos` и `assets/icons` - логотипы и иконки.
- `assets/docs` - PDF и другие документы для скачивания.
- `styles` - разделенные CSS-слои: токены, база, layout, компоненты, секции, responsive.
- `scripts` - модульный JavaScript по зонам ответственности.
- `data` - нормализованные JSON-данные по разделам и контентным блокам.

## Именование файлов

- Только латиница.
- Нижний регистр и `kebab-case`.
- Без пробелов и случайных служебных названий.

## Что где менять

- Тексты и структура страницы: `index.html`.
- Общие цвета, радиусы, тени, контейнер: `styles/tokens.css`.
- Базовая типографика и reset: `styles/base.css`.
- Общий layout, header, scrollers: `styles/layout.css`.
- Переиспользуемые UI-компоненты и модалки: `styles/components.css`.
- Секции страницы и фоновые медиа-классы: `styles/sections.css`.
- Адаптив: `styles/responsive.css`.
- Поведение шапки: `scripts/header.js`.
- Модалки: `scripts/modals.js`.
- Галерея: `scripts/gallery.js`.
- Секционные интеракции и scrollers: `scripts/sections.js`.
- Внешние виджеты и мелкие тех. улучшения: `scripts/external.js`.

## Данные

Сейчас страница остается статической, но в `data/*.json` уже есть нормализованные данные для будущего перехода к шаблонной генерации повторяющихся блоков.
